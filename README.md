Selamat datang di **Spiders**. Repository ini adalah pusat dari portofolio profesional saya sekaligus wadah komunitas bagi para antusias teknologi, keamanan siber, dan sistem komputer. 

Proyek ini dibangun untuk mendokumentasikan perjalanan akademik saya di **Universitas Tanjungpura (UNTAN)** serta eksplorasi saya dalam dunia *Hacking* dan *Algorithmic Trading*.

---

## 🚀 Key Features

- **Personal Portfolio:** Showcase proyek hardware, software, dan sertifikasi.
- **Cybersecurity Lab:** Dokumentasi hasil belajar *penetration testing* dan *CTF write-ups*.
- **Trading Journal:** Integrasi data atau strategi trading yang sedang dipelajari.
- **Community Space:** Area diskusi untuk kolaborasi antar sesama mahasiswa atau praktisi.

## 🛠️ Tech Stack

- **Frontend:** Next.js / React.js (High Performance)
- **Styling:** Tailwind CSS (Modern UI)
- **Environment:** Windows for development, Kali Linux for security testing.
- **Deployment:** Vercel / Netlify.
